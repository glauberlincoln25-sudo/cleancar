MANUAL DE INSTRUÇÕES
Guia completo do sistema v4.0
🔐 ACESSO E LOGIN
1
Acessar o sistema: Abra glauberlincoln25-sudo.github.io/cleancar no navegador.
2
Login: Digite e-mail e senha → clique ▶ Entrar ou pressione Enter.
3
Indicador de conexão: 🟢 Firebase online · 🔴 offline (dados salvos localmente até reconectar).
4
Sair: Clique em Sair no topo direito a qualquer momento.
💡 Para criar novos usuários: Firebase Console → Authentication → Users → Add User.
🎨 PERSONALIZAÇÃO DA INTERFACE
1
Logo: Admin → Empresa → upload do arquivo PNG/JPG.
2
Cor principal: Admin → Empresa → selecione ou use o color picker → Salvar Aparência.
3
Tema escuro/claro: Clique no botão 🌙/☀️ no topo da tela.
4
Busca global: Pressione Ctrl+K para buscar clientes, serviços e agendamentos.
👥 CADASTRAR CLIENTE
1
Menu → Cadastro → preencha Nome, Telefone e demais dados.
2
CEP automático: Digite o CEP → clique 🔍 ou pressione Enter → endereço preenchido.
3
Clique ✓ Salvar Cliente. O cliente aparece no Banco de Dados.
4
Editar: Banco de Dados → botão ✎ na linha do cliente.
5
Histórico completo: Banco de Dados → botão 📋 → veja todos os serviços, total gasto e primeira visita.
🔲 QR CODE DE FIDELIDADE
1
Menu → Fidelidade → botão 🔲 QR Fidelidade.
2
Selecione o cliente → QR Code é gerado automaticamente.
3
Clique 🖨 Imprimir QR → entregue ao cliente como adesivo ou impresso.
4
Ao escanear o QR, o sistema abre a ficha do cliente diretamente.
📅 CRIAR AGENDAMENTO
1
Menu → Agendamento → busque o cliente por nome, CPF ou telefone.
2
Selecione a data, horário e veículo. O horário de fim é calculado automaticamente.
3
Clique ＋ Adicionar Serviço → selecione os serviços (pode ser mais de um).
4
Aplique desconto se necessário (requer senha ADM se acima do limite).
5
Clique ✓ Confirmar Agendamento.
✓ Ao concluir um agendamento: financeiro, estoque, fidelidade e NPS são atualizados automaticamente.
✓ CHECKLIST COM ASSINATURA
1
Menu → Checklist → selecione o agendamento vinculado.
2
Marque cada item da vistoria como OK ou com problema.
3
Fotos: Clique 📷 para tirar foto do veículo com a câmera do celular.
4
Assinatura digital: O cliente assina diretamente na tela com o dedo ou mouse.
5
Clique ✓ Aprovar e Assinar → assinatura + fotos salvos no registro.
🖨 ORDEM DE SERVIÇO (OS)
1
Agenda Geral → botão 🖨 na linha do agendamento.
2
A OS abre em nova janela com QR Code, dados do cliente, veículo e valor.
3
Clique 🖨 Imprimir ou use Ctrl+P.
💵 CAIXA DO DIA
1
Abrir caixa: Menu → Caixa do Dia → clique ▶ Abrir Caixa.
2
Serviços concluídos são lançados automaticamente no caixa.
3
Entradas/saídas manuais: Clique ＋ Entrada ou ＋ Saída.
4
Fechar caixa: Clique ■ Fechar Caixa → confirme o fechamento.
5
Imprimir fechamento: Clique 🖨 para gerar o relatório do dia.
📊 RELATÓRIOS E LUCRO LÍQUIDO
1
Menu → Relatórios → veja faturamento, formas de pagamento e top serviços.
2
Lucro Líquido Real: Configure impostos, comissões e custos fixos clicando em ⚙ Configurar custos.
3
NPS: Veja as notas e comentários dos clientes — promotores, neutros e detratores.
4
Período: Alterne entre Diário, Mensal e Anual pelo seletor no gráfico.
📦 GERENCIAR ESTOQUE
1
Menu → Estoque → ＋ Adicionar Produto.
2
Informe nome, categoria, unidade (ml/g/unid), volume por embalagem e quantidade.
3
O total disponível é calculado automaticamente (embalagens × volume).
4
Alertas: Produtos abaixo do mínimo aparecem com ⚠ em vermelho.
5
Baixa automática: Ao concluir um serviço, os insumos vinculados são debitados automaticamente.
💧 SANGRIA DE INSUMO (PERDAS)
1
Estoque → botão 💧 Sangria de Insumo.
2
Selecione o produto, informe a quantidade perdida e o motivo.
3
Clique 💧 Registrar Sangria → produto debitado + lançado como saída no financeiro.
🧮 FICHA TÉCNICA
1
Menu → Ficha Técnica → configure custos fixos e mão de obra.
2
Selecione o serviço → adicione os insumos usados com as quantidades.
3
O sistema calcula o custo real e sugere um preço com a margem desejada.
4
Clique ↑ Cadastrar valor final no Serviço para aplicar o preço calculado.
⭐ PROGRAMA DE FIDELIDADE
1
Criar regra: Fidelidade → ⚙ Gerenciar Regras → ＋ Nova Regra.
2
Defina os pontos necessários, benefício (desconto % ou serviço grátis) e descrição.
3
Editar regra: Botão ✎ na linha da regra → altere e salve.
4
Excluir regra: Botão ✕ na linha da regra → confirme.
5
Pontos automáticos: São adicionados ao concluir serviço (configure em Admin → Regras).
6
Resgatar benefício: Ranking → botão 🎁 Resgatar na linha do cliente.
💬 CONFIGURAR WHATSAPP
1
Menu → WhatsApp → aba ⚙ API.
2
A URL e chave já estão pré-configuradas. Clique 🔌 Testar Conexão.
3
Se não conectado, clique 📱 Conectar via QR Code → escaneie com o WhatsApp do celular.
4
Após conectar: o indicador fica 🟢 e o botão mostra ⚡ Enviar via API.
📤 ENVIAR MENSAGEM
1
WhatsApp → 📤 Enviar Mensagem → busque o cliente.
2
Selecione um template da lista → preview aparece automaticamente com dados do cliente.
3
Vincule um agendamento (opcional) para preencher data, hora e valor.
4
Clique ⚡ Enviar via API (direto) ou 💬 WhatsApp (abre o app).
📝 CRIAR TEMPLATE
1
WhatsApp → 📝 Templates → preencha nome, categoria e mensagem.
2
Use variáveis clicáveis: {nome} {data} {hora} {servico} {valor} etc.
3
O preview mostra a mensagem com dados de exemplo em tempo real.
4
Clique ✓ Salvar Template. Edite clicando em ✎ na tabela.
📢 ENVIO EM LOTE
1
WhatsApp → 📢 Envio em Lote → selecione o template e o filtro.
2
Filtros disponíveis: Todos, Inativos +90 dias, Agendados amanhã, Aniversariantes, Com pontos.
3
Confirme a lista de destinatários → clique 💬 Iniciar Envio em Lote.

Edite o HTML/JS no editor — use as funções do sistema como dbGet, toast, goTo.
4
Clique 👁 Preview para testar antes de criar.
5
Clique ＋ Criar Tela → aparece no menu lateral automaticamente.
